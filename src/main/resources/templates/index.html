Growth Parameter Chart

Code Sample (Spring Boot with JFreeChart to generate growth chart images)

The following chart is stature percentile based on age for my daughter. If you want different REST service call, use the following:
ageInMonth--> age in month, height--> height in centimeter, sex--> 1:boy 2:girl)
http://localhost:8080/chart/{ageInMonth}/{height}/{sex}
Examples:
.../chart/160/156/2 (a girl who is 160 months old with 156 centimeter height)
.../chart/150/155/1 (a boy who is 150 months old with 155 centimeter height)
